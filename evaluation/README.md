# Evaluation Scaffold for SmartDoc AI

Thu muc nay dung de chay benchmark va luu ket qua phuc vu bao cao IEEE.

## Cau truc de xuat
- `dataset/`: tap cau hoi benchmark va nhan tham chieu.
- `results/`: file CSV/JSON sinh ra sau moi lan chay.
- `plots/`: bieu do PNG/SVG de chen vao bao cao.
- `scripts/`: script chay PaCRAG/GraphRAG va tong hop so lieu.

## Muc tieu
- So sanh PaCRAG voi GraphRAG tren cung mot tap cau hoi.
- Do latency, retrieval quality, va cac proxy metric co san.
- Tao dau ra lap lai duoc de doi chieu voi noi dung trong `IEEECS_SGU_2026/report_final.tex`.

## Ghi chu
- Cac so lieu trong bao cao chi nen lay tu ket qua trong thu muc nay.
- Neu chua co du lieu, giu nhan `proxy` hoac `preliminary`.
